Tasky

Tasky is a simple task management app to organize your ToDo lists.

Author : Manmeet Singh

IzzySoft
GitHub

Tasky is a highly intuitive and easy-to-use todo list application designed for those who seek a powerful yet minimalistic way to manage their daily tasks.

Features

Minimal Design

Minimal design of the app makes it the easy to stay distraction free!

Open Source

The code of the app is completely public and anyone can view and modify it.

Beautiful UI

Follows the latest material 3 guidelines from google providing with beautiful UI.

Material YOU

Adaptive colors for android devices running 12+ version according to the wallpaper.

Reminders

Get Reminders according to the date and time selected while adding a task, so you don't miss on some important task!

Animations and Sounds

Classic Task completion and Deletion sounds with confetti animations for completing a task!.
